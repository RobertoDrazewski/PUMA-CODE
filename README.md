🛠️ Getting Started
Follow these steps to get the project running locally on your machine.

1. Prerequisites
Ensure you have Node.js (v18 or higher) and npm installed.

2. Installation
Clone the repository and install the necessary dependencies:

Bash
git clone https://github.com/RobertoDrazewski/PUMA-CODE
cd puma-code
npm install
3. Execution
Launch the development server:

Bash
npm run dev
Open http://localhost:3000 in your browser to see the result.

4. Build for Production
To create an optimized production build:

Bash
npm run build
npm run start
📱 Mobile App Setup (PWA)
The project is configured to display a custom icon when a user adds the site to their phone's home screen.

Icons: * Place your 180x180px logo in public/apple-icon.png (for iOS).

Place your 192x192px logo in public/icon-192x192.png (for Android/Chrome).

Manifest: Customize public/manifest.json to change the app name or theme colors.

Testing: Open the site on Safari (iOS) or Chrome (Android) and select "Add to Home Screen".
